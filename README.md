# Home Workout Coach V2

A phone friendly Expo React Native fitness app for bodyweight workouts.

## Features

- Home, Calendar, History, and Settings tabs
- adjustable difficulty: Beginner, Intermediate, Advanced
- workout focus areas: Full Body, Abs, Chest, Legs, Arms, Butt
- adaptive daily workout plan based on feedback
- post workout feedback: Too Easy, Just Right, Too Hard
- monthly calendar of completed sessions
- workout history page
- weekly goal tracking
- optional online workout suggestion endpoint
- local saving with AsyncStorage

## Run

```bash
npm install
npx expo start
```

Then open it with Expo Go on iPhone.

## Notes

This app works offline with a built in adaptive engine.

If you later host a backend endpoint, you can paste that URL into the Settings tab and enable online suggestions.
